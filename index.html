Pros and cons of application testing

Pros

  • reduce the number of regressions
  • give more confidence when refactoring
  • resolve issues when updating dependencies

Cons

  • testing requires additional financial costs
  • we need more time in the initial stages
  • it is necessary to think over the structure for tests
